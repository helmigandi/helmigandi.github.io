Remove Linux Dual Boot With Windows MBR Partition

May 26, 2020

How to remove Linux Operating System Dual Boot with Windows Operating System With MBR Partition. Here are the steps:

1. Install Mini Tool Partition Wizard#

  • Download free at Partition Wizard Website.

2. Check whether our disk uses MBR or not#

  • Open Disk Management in Windows search.
  • Right click on our Hard drive (usually Hard drive 0).
  • Select Properties and click the Volumes section Tab.
  • Look at the picture below:

3. Do The Partition#

  • In the Mini Tool Partition Wizard program, select Disk & Partition Management. Following pictures and explanations below:

Steps:

  1. Select the partition containing Linux (mine is partition 3). Right click and select Delete.
  2. Select Disk whose partition was in Delete earlier (mine is disk 1). Right click and select Rebuild MBR.
  3. Operations Pending which contains information on the flow that we have done, namely Delete partition and Rebuild MBR, but we can still Undo or not.
  4. Apply for final steps.
